## 2. Tracker réglementaire — Pipeline

### Architecture
```
earlybrief-platform (Scaleway)
→ GET /api/v1/articles (published_at, score, source)
→ tracker-admin (early-brief.com/outils/tracker-admin/)
→ validation humaine
→ describe-tracker.js (reformulation + classification Claude)
→ tracker-data.json + autres-publications-data.json
→ GitHub API commit
→ Netlify deploy automatique
```

### Filtres tracker-admin
- `isTrackerSource` : PRIORITY_SOURCES (tout passe) + OTHER_SOURCES (score >= 30)
- `isNotGoogleNews` : exclut news.google.com
- `isOfficialDomain` : 30 domaines officiels whitelistés
- `isNotGeneric` : filtre titres génériques (< 15 chars ou dans GENERIC_TITLES)
- `isNew` : déduplication contre entrées existantes

### PRIORITY_SOURCES (score ignoré, tout passe)
AMLA, EBA, ESMA, AMF, ACPR, Tracfin, FATF, Conseil UE, Commission EU, OFAC, OFSI, Moneyval, FinCEN, Legifrance

### OTHER_SOURCES (score >= 30)
Egmont, Wolfsberg, BIS, FSB, EIOPA, ECB, CNIL, AI Office
